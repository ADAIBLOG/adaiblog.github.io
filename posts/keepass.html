密码管理器：Keepass系列

阿呆2025-01-242026-02-25

KeePass (https://keepass.info/) 是桌面端的原版和官方项目，提供标准数据库文件的技术文档。它定期更新，并处于积极维护状态（使用 C# 编写）。
KeePassXC (https://keepassxc.org/) 是用 C++ 编写的一个 KeePass 的替代集成版本。
KeeWeb (https://keeweb.info/) 是一个网页版本，也兼容 KeePass 文件。
KeePassDX (https://www.keepassdx.com/) 是一个安卓软件
KeePass2AnDroid (https://github.com/PhilippC/keepass2android) 是一个安卓软件

KeePass

KeePassXC 是一款开源的密码管理软件，支持本地离线存储密码数据库，通过高强度加密（如 AES-256）保护用户数据。与云端密码管理器不同，KeePass 的数据库文件（.kdbx）完全由用户掌控，适合注重隐私与安全的人群。

创建密码数据库

  1. 新建数据库
    打开 KeePass，点击 File → New。
    设置保存路径和文件名（例如 MyPasswords.kdbx）。
  2. 设置主密码与密钥文件
    主密码：输入一个高强度密码（建议 12 位以上，含大小写字母、数字、符号）。
    密钥文件（可选）：可额外绑定一个文件（如 U 盘中的文件）作为双重验证。

    重要提示：主密码和密钥文件丢失将无法恢复数据库！建议备份至安全位置。

  3. 配置加密设置
    默认使用 AES-256 加密，保持默认设置即可。
    点击 OK 完成创建。

新建密码条目

  1. 右键左侧目录 → Add Entry。
  2. 填写信息：
    • Title：网站/应用名称（如“Gmail 账户”）。
    • User name：用户名或邮箱。
    • Password：点击生成按钮生成随机密码（可自定义长度和字符类型）。
    • URL：关联的网站链接（可选）。
    • Notes：备注信息（如安全问题答案）。

配置浏览器集成

  1. 打开KeePassXC->工具->设置->浏览器集成
  2. 浏览器点击KeePassXC-Browser 扩展图标，点击连接按钮
  3. 输入连接设备名称点击保存并允许访问

自动输入规则

编辑条目->自动输入->使用自定义自动输入序列

{USERNAME}用户名

{PASSWORD}用户密码

{TOTP}TOTP值

{DELAY 2500}2500毫秒延迟

{TAB},{ENTER},{SPACE}对应键盘按键功能

{CLEARFIELD}清除输入字段

{CLEARFIELD}{USERNAME}{TAB}{PASSWORD}{ENTER}

{CLEARFIELD}{USERNAME}{TAB}{PASSWORD}{ENTER}{DELAY 2500}{TOTP}{ENTER}

{CLEARFIELD}{USERNAME}{ENTER}{DELAY 2500}{PASSWORD}{ENTER}{ENTER}

{CLEARFIELD}{USERNAME}{ENTER}{DELAY 2500}{PASSWORD}{ENTER}{DELAY 2500}{TOTP}{ENTER}

快速打开数据库

再应用首页点击需要快速打开的数据库右边的⭐号，下次打开软件自动打开这个数据库
![](/img/posts/keepass/KeepassDX (1).jpg)

启动指纹验证

  1. 点击设置 -> 设备解锁 -> 勾选生物识别解锁,自动打开提示
    []
  2. 输入密码，点击左下角设备解锁链接
    ![](/img/posts/keepass/KeepassDX (5).jpg)
  3. 解锁成功之后，后续打开数据库自动弹出指纹解锁弹窗直接解锁数据库

    设备凭据解锁就是使用解屏密码
