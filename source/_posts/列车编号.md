# C(城)
城际动车组旅客列车
# D(动)
动车组旅客列车
# G(高)
高速动车组旅客列车
# K(快)
快速旅客列车
# L(临)、A
临时旅客列车
# N (内)
管内快速列车
# S(市)
市郊列车
# T(特)
特快旅客列车
# X(行)
行包列车
# Y(游)
旅游列车
# Z(直)
直达特快列车
# 无字母
普通旅客列车
